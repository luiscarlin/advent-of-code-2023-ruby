# Advent of Code 2023 Ruby

My solutions to Advent of Code 2023 in Ruby

## Prerequisites

- [rvm](https://rvm.io/)

## Setup

```bash
# use correct version
rvm use

# install gems
bundle install

# start day
./bin/start day04

```

## Run

```bash
# run
ruby ./day01/day01.rb

# linting
bundle exec rubocop

```
